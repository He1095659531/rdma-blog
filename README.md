# RDMA驱动学习

## 写在前面

随着AI应用的火爆，越来越多的相关从业人员将目光投入到可显著提升AI训练和推理效率的关键技术——RDMA上，其在底层网络通信技术上的重要性被越来越多人所熟知。从最初只有Mellanox生产的CX系列卡件到现在国内海思的hns系列网卡，越来越多的头部公司开始投入RDMA技术的研发。在AI崛起带来的时代风口上，RDMA技术已经在底层网络通信技术中占据主流。在这样的背景下，笔者翻遍网络发现除了[RDMA圣经](https://zhuanlan.zhihu.com/p/164908617)外，其他的RDMA相关博客要么零散，要么不够深入，并不能帮助新人更好的入门。

为了让更多的开发者能够对RDMA技术有更清晰的了解，我会基于我的工作经验以及他人的博客，为大家带来一系列RDMA文字，主要涵盖用户态与内核态交互、RDMA驱动定制化修改、CM建立流程、OFED与驱动交互流程，还有驱动与硬件交互以及Reg MR流程中的MTT表的实现等多方面内容。
